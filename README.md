# yami-streamlit

## study streamlit
Reference link: [streamlit doc](https://docs.streamlit.io/)

## How to run?

### step 1 - create python environment venv

```commandline
python -m venv .venv
```
### step 2 - install python dependencies

```commandline

pip install -r requirements.txt
```

### step 3 - run the project
```commandline
streamlit run main.py
```
